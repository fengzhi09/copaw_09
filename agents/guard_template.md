# Agent 安全检查清单

## 1. 身份认证

- [ ] 验证用户身份
- [ ] 检查 API Key / Token 有效性
- [ ] 确认用户权限

## 2. 输入验证

- [ ] 检查输入内容是否为空
- [ ] 过滤敏感词
- [ ] 限制输入长度
- [ ] 防止 SQL 注入
- [ ] 防止 XSS 攻击

## 3. 输出过滤

- [ ] 敏感信息脱敏
- [ ] 过滤敏感词
- [ ] 检查输出内容合规性

## 4. 资源限制

- [ ] 检查 API 调用频率
- [ ] 限制文件大小
- [ ] 限制内存使用
- [ ] 超时保护

## 5. 操作审计

- [ ] 记录操作日志
- [ ] 保存审计轨迹
- [ ] 异常行为告警

## 6. 网络安全

- [ ] HTTPS 加密传输
- [ ] 验证外部链接
- [ ] 防止 CSRF 攻击

## 7. 数据安全

- [ ] 敏感数据加密存储
- [ ] 定期备份数据
- [ ] 数据访问控制

## 8. 合规检查

- [ ] 符合隐私政策
- [ ] 符合服务条款
- [ ] 版权检查
