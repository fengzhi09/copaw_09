# 00 号管理高手 - 安全检查清单

## 1. 身份认证

- [x] 验证用户身份（allowFrom 白名单）
- [x] 检查 API Key / Token 有效性
- [x] 确认用户权限（仅管理员可操作）

## 2. 输入验证

- [x] 检查输入内容是否为空
- [x] 过滤敏感词
- [x] 限制输入长度（防止恶意构造）
- [x] 防止 SQL 注入（参数化查询）
- [x] 防止 XSS 攻击（输出转义）

## 3. Agent 创建安全

- [x] 创建前需要用户确认
- [x] 限制 Agent 数量（05-08 预留）
- [x] 检查配额配置合理性
- [x] 防止创建恶意 Agent

## 4. 操作审计

- [x] 记录所有创建/修改操作
- [x] 保存审计轨迹
- [x] 异常行为告警（通知用户）

## 5. 敏感操作确认

- [x] 创建 Agent 需确认
- [x] 删除 Agent 需确认
- [x] 修改配额需确认
- [x] 删除记忆需确认

## 6. 数据安全

- [x] 配置文件加密存储
- [x] API Key 加密存储

## 7. 网络安全
